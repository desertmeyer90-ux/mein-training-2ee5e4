# Nutzung direkt im Browser (ohne Installation)

Für alle, die nichts installieren möchten, oder am Laptop/PC testen.

1. Öffne [TEST-URL] in einem aktuellen Browser (Chrome, Safari, Edge oder Firefox).
2. Durchlaufe die Einrichtung. Fertig.
3. **Lesezeichen setzen**, damit du die Adresse wiederfindest.

## Wichtige Unterschiede zur installierten App

- Deine Daten hängen an **genau diesem Browser auf genau diesem Gerät**. Anderes Gerät oder anderer Browser = leere App (dafür gibt es Backup + Import).
- **Privat-/Inkognito-Modus vermeiden:** dort werden die Daten beim Schließen gelöscht!
- Browserdaten/Websitedaten löschen = Trainingsdaten weg. Vorher immer „Mehr → Backup speichern".
- Offline sollen Training, Pläne, Fotos und Verlauf auch im Browser funktionieren, sobald die Seite einmal vollständig geladen war (gleiche Adresse aufrufen). Grenzen: erstes Laden und Updates brauchen Internet.
- „Bildschirm bleibt an" im Training funktioniert je nach Browser/Gerät unterschiedlich zuverlässig; auf dem Handy ist die installierte Variante besser.

Backup: Die .json-Datei landet in deinem Download-Ordner. Sicher aufbewahren.
